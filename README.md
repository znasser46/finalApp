# finalApp
## App Idea:
### This app will allow users to save, edit and delete budgets based on their monthly spending amounts that they enter. It will also calculate their total discretionary income.

## Target User:
### People trying to manage their expenses with a detailed record of new and past budgets.

## Features:

* User Authentication
*  Create, Read, Update, Delete implementation
*  Modern node structure
*  Mongo Database connection
*  seach/tagging
*  Monthly spending totals

## Tech Stack:
* HTML
* CSS
* Bootstrap
* Javascript
* Node.js
* Express
* MongoDB
* GCP
* Nginx
* Pm2
